## Installation üíæ
```
git clone https://github.com/Aninimo/doctor-dashboard.git
```

## Setup .env file
```
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=
NEXT_PUBLIC_CLERK_SIGN_IN_URL=/sign-in
NEXT_PUBLIC_CLERK_SIGN_UP_URL=/sign-up
```

## Install deps:
```
npm install
```

## Run dev server:
```
npm run dev
```
